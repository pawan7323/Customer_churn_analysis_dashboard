# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview

The **Customer Churn Analysis Dashboard** is an interactive Power BI project designed to analyze customer behavior and identify the factors contributing to customer churn. The dashboard helps businesses understand customer retention trends, monitor key performance indicators (KPIs), and make data-driven decisions to reduce churn.



## 🎯 Objectives

- Analyze customer churn patterns.
- Identify high-risk customers.
- Compare churn across different customer segments.
- Monitor important business KPIs.
- Support strategic decision-making for customer retention.

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- **Microsoft Excel / CSV Dataset**
- **Data Modeling**

## 📌 Business Insights

Some important insights generated from the dashboard include:

- Customers with **Month-to-Month contracts** have a higher churn rate.
- Customers paying through **Electronic Check** tend to churn more frequently.
- Higher **Monthly Charges** are associated with increased churn.
- Customers with shorter **tenure** are more likely to leave.
- Long-term contract customers show better retention.
